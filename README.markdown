# haroog 
>haroog 는 FRENDS 내부 해카톤 행사 참여를 위한 프로젝트입니다.

haroog 는 node.js 기반으로 작성된 static 블로그 엔진입니다.

문서 작성은 markdown 포맷만 지원하며 블로그 ui 는 twitter bootstrap 을 이용하였습니다.
